# Bank Customer Churn Analysis

## Project Overview
This project analyzes customer churn behavior in a banking dataset using Python, SQL, and Tableau.

## Tools Used
- Python
- Pandas
- NumPy
- SQL
- Tableau
- Google Colab

## Project Workflow
1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. SQL-Based Analysis
5. Customer Segmentation
6. Visualization
7. Business Insights
8. Recommendations

## Tableau Visualizations
- Geography vs Churn
- Gender vs Churn
- Age Group vs Churn
- Active vs Inactive Members
- Credit Score Group vs Churn
- Balance Category vs Churn

## Business Insights
- Germany has the highest churn rate.
- Senior customers are more likely to churn.
- Inactive members show higher churn behavior.
- Customers with low credit scores are high-risk customers.

## Recommendations
- Improve retention campaigns for senior customers.
- Increase engagement for inactive members.
- Provide personalized offers to high-risk customers.
